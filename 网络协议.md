# 网络协议

## 常见的 HTTP Code

- 1XX - 服务器已收到请求，正在处理，但还未完成
- 2XX - 请求成功，服务器已成功返回数据
- 3XX - 需要客户端采取进一步操作（如跳转）
- 4XX - 客户端请求错误，服务器无法处理
- 5XX - 服务器内部出错，无法完成请求
