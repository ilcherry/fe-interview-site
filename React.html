React 的重渲染 ​

  1. 组件的 state 变化
  2. 父组件重渲染导致子组件也重渲染
  3. 组件接收到新的 props
  4. context 变化导致所有消费组件重渲染

在 React 渲染列表时，需要为每个列表项添加 key，其主要原因是 提升性能并减少不必要的 DOM 操作
